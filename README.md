# 🎬 Movie Watchlist App

## 📌 About the Project

This is a simple movie web app built as part of my individual project. The idea is to let users search for movies and keep track of what they want to watch.

The app uses a public API to fetch real-time movie data and displays it in a clean and interactive way.

---

## 🌐 API Used

OMDb API
https://www.omdbapi.com/

---

## ⚙️ Features (Planned)

* Search movies by name
* Display movie results dynamically
* Add/remove movies from a watchlist
* Sort movies (year/title)
* Filter by type (movie/series)

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Fetch API
